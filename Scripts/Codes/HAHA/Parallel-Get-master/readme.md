Parallel Get
-------

*Parallel Get*
是一个基于MATLAB 平台，用于获取股票历史/即时行情数据的工具。

最低版本支持:2014b

----
使用方法
=========
下载并添加至Path 即可直接调用函数。

----------

API 详解
====
**get_tick_data（code，date）**
获取个股以往某日历史的分笔数据明细。

示例1：
>\>\>get_tick_data('000002','2016-08-26')
>

    '14:56:48'    '23.57'    '--'       '1428'     '3367681'
    '14:56:45'    '23.57'    '0.02'     '350'      '824950'
